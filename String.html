1/ String(chuỗi) trong Javascript

String là 1 đoạn text có thể có một hoặc nhiều kí tự. Thông thường ta lưu nó vào 1 biến và biến này được gọi là biến kiểu dữ liệu Chuỗi. Tất cả các chuỗi đều phải được bao quanh bằng kí tự nháy đơn ' ' hoặc nháy đôi " ".

Ví dụ : var varName = "Đây là 1 đoạn text có kiểu dữ liệu là Chuỗi"

Trong trường hợp có dấu nháy đơn hoặc nháy đôi xuất hiện trong chuỗi thì bắt buộc phải thêm kí tự \ đằng trước dấu nháy đó để không bị lỗi cú pháp và được hiểu đó là nội dung của đoạn text.

var stringName = "This is a string text or you can call : \"Chuỗi\" "

2/ Nối chuỗi trong Javascript

Để nối chuỗi ta chỉ đơn giản là dùng dấu + để nối các chuỗi lại với nhau.

Ví dụ : varName = "Chào" + "Mừng" + "Bạn";

varName2 = "Đến" + "Đây";

varName3 = varName + varName2;

Trường hợp khi xuống hàng trong chuỗi(nhấn enter xuống hàng) trong Javascript thì bắt buộc phải dùng dấu + nối chuỗi nếu không sẽ bị lỗi cú pháp.

var varName4 = "Chào mừng bạn đã đến "

+ "với Javascript.";

Có 1 cách khác viết gọn hơn là dùng dấu \ để trình duyệt biết là có xuống hàng.

var varName5 = "Chào mừng bạn lại đến \

với Javascript lần nữa"

3/ Ép sang kiểu chuỗi String trong javascript

Để ép 1 kiểu chuỗi nào đó sang kiểu String thì sử dụng cú pháp string.toString()

Ví dụ:

// Trước khi chuyển đổi

var number = 12;

alert(typeof number);

// Sau khi chuyển đổi

number = number.toString();

alert(typeof number);

Từ khóa typeof vars sẽ trả về kiểu dữ liệu của biến vars.

Ngoài cách trên đây bạn có thể sử dụng String để tạo 1 chuỗi và vì là đối tượng nên nó phải có từ khóa new phía trước

Ví dụ:

var msg = new String("Đây là 1 kiểu String");

Lúc này khi bạn truyền vào kiểu gì đi nữa thì kết quả trả về nó vẫn là kiểu String. Tuy nhiên không nên sử dụng cách này bởi vì chương trình sẽ chạy chậm hơn.
