1.Thêm sự kiện cho một thẻ HTML bằng javascript

Cú pháp như sau :

elementObject.eventName = function() { ... };

Trong đó:
- elementObject là đối tượng HTML mà chúng ta sử dụng DOM để lấy.
- eventName là tên của event như onclick, onchange, ...



2.Thêm sự kiện cho nhiều thẻ HTML bằng javascript

Ở trên mình sử dụng phương thức document.getElementById() nên kết quả trả về chỉ có 1 đối tượng và vì thế có thể gắn trực tiếp varName.eventName = function(){}

Nhưng nếu sử dụng truy vấn DOM theo selector thì kết quả trả về sẽ là 1 mảng các đối tượng và lúc này bạn phải sử dụng vòng lặp để thiết lập sự kiện.

Cú pháp như sau :

Lấy đối tượng html, bạn có thể sử dụng các DOM Element khác để lấy

var varName = document.getElementsByTagName('element');

Lặp qua từng phần tử trong kết quả và gán sự kiện

for(var i = 0; i < varName.length; i++){

varName[i].eventName = function(){

Do Something

};

}
