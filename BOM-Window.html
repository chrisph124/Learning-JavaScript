1/ Thao tác với 1 cửa sổ window

Đối tượng Window cung cấp 1 số phương thức giúp thao tác với trình duyệt dễ dàng hơn.

a/ Mở 1 cửa sổ với lệnh window.open()

Cú pháp: window.open(url,name,options)

Trong đó:

url: là đường dẫn website muốn mở

name: là tên bạn đặt cho cửa sổ này

options: là 1 chuỗi các thông số được cách nhau bởi dấu phẩy, sau đây là 1 số thông số thông dụng:

height = pixels hiển thị chiều cao của cửa sổ.

width = pixels hiển thị chiều rộng của cửa sổ.

top = pixels hiển thị lề trên so với cửa sổ.

left = pixels hiển thị lề trái so với cửa sổ.

menubar = yes | no | 1 | 0 có/không hiển thị thanh menu.

resizable = yes | no | 1 | 0 có/không hiển thị biểu tượng resize cửa sổ.

scrollbars = yes | no | 1 | 0 có/không hiển thị thanh cuộn cửa sổ.

status = yes | no | 1 | 0 có/không hiển thị thanh trạng thái.

titlebar = yes | no | 1 | 0 có/không hiển thị titlebar

toolbar = yes | no | 1 | 0 có/không hiển thị thanh toolbar

fullscreen = yes | no | 1 | 0 có/không hiển thị biểu tượng fullscreen cửa sổ



Ví dụ:Open



b/ Đóng 1 cửa sổ với lệnh window.close()

Sau khi mở 1 cửa sổ ra thì để đóng nó ta sử dụng lệnh windowObj.close(). Trong đó windowObj là cửa sổ mà ta sử dụng lệnh window.open() tạo ra

c/ Di chuyển cửa sổ với lệnh window.moveTo()

Sau khi mở 1 cửa sổ, nếu muốn di chuyển nó thì ta dùng lệnh window.moveTo(top,left). Trong đó:

Top: là khoảng cách cửa sổ so với lề trên ( đơn vị : pixels)

Left: là khoảng cách cửa sổ so với lề trái ( đơn vị : pixels)



Ví dụ:

Open

Move

Close
