1/ Đối tượng(object) là gì ?

Như ta biết Javascript không được xếp vào danh sách những ngôn ngữ lập trình hướng đối tượng, tuy nhiên về khách quan mà nói thì bạn hoàn toàn có thể lập trình hướng đối tượng thông qua kiểu dữ liệu Object mà Javascript đã cung cấp

Trong Javascript, đối tượng là 1 khái niệm trừu tượng thể hiện cho 1 đối tượng cụ thể và nó có sẵn 1 số đối tượng như Date, Number, ...

Ngoài các đối tượng trên thì ta vẫn có thể tạo ra 1 đối tượng theo ý mình dựa theo vào yêu cầu của ứng dụng.

a/ Khởi tạo đối tượng

Sử dụng từ khóa new Object()

var Comment = new Object();

Sử dụng từ khóa {}

var Comment = {};

2/ Thuộc tính và phương thức của đối tượng

a/ Thuộc tính

Là những đặc điểm( có thể hiểu là 1 biến ) cần lưu trữ trong đối tượng. Ví dụ với đối tượng Comment thì cần có những thuộc tính sau :

  • title
  • content
  • fullname
  • email

Có 3 các để khai báo như sau :

Cách 1 : sử dụng từ khóa new Object()

//Khởi tạo

var Comment = new Object();

//Thêm thuộc tính

Comment.title = '';

Comment.content = '';

Comment.fullname = '';

Comment.email = '';

Cách 2 : sử dụng từ khóa {} và thêm thuộc tính ngay lúc khai báo

//Khởi tạo và thêm thuộc tính

var Comment = {

title = '',

content = '',

fullname = '';

email = ''

};

Cách 3 : sử dụng từ khóa {} và thêm thuộc tính sau đó

//Khởi tạo

var Comment = {};

//Thêm thuộc tính

Comment.title = '';

Comment.content = '';

Comment.fullname = '';

Comment.email = '';

Trong 3 cách này thì cách 2 là cách tối ưu nhất vì nó mạch lạc, dễ hiểu và dể quản lý code.

Phương thức

Nếu như Thuộc tính là những đặc điểm cần lưu trữ trong đối tượng thì Phương thức là 1 hành động( có thể hiểu là 1 hàm ) của đối tượng.

Ví dụ trong đối tượng Comment ta cần 2 phương thức là :

  • addComment()
  • validateComment()

Và vẫn sẽ có 3 cách khai báo như sau:

Cách 1 : sử dụng từ khóa new Object()

//Khởi tạo

var Comment = new Object();

//Thêm phương thức

Comment.addComent = function (){

//Hành động nào đó

}

Comment.validateComment = function(){

//Hành động nào đó

}

Cách 2 : sử dụng từ khóa {} và thêm thuộc tính ngay lúc khai báo

//Khởi tạo và thêm thuộc tính

var Comment = {

addComment = function(){

//Hành động nào đó

},

validateComment = function(){

//Hành động nào đó

}

};

Cách 3 : sử dụng từ khóa {} và thêm thuộc tính sau đó

//Khởi tạo

var Comment = {};

//Thêm phương thức

Comment.addComment = function(){

//Do Something

};

Comment.validateComment = function(){

//Do Something

};

Cách 2 vẫn là cách tối ưu nhất vì nó mạch lạc, dễ hiểu và dể quản lý code.

3/ Thao tác với thuộc tính và phương thức trong đối tượng

Sau khi tạo xong đối tượng thì có 2 cách sử dụng căn bản là gọi và gán dữ liệu cho thuộc tính và phương thức.

Nhưng thao tác cơ bản vẫn là :

  • Gán giá trị cho thuộc tính
  • Lấy giá trị của thuộc tính
  • Gọi phương thức

a/ Gán giá trị cho thuộc tính

Để gán giá trị cho thuộc tính thì ta chỉ cần sử dụng toán tử = giống như cách gán giá trị cho biến

Comment.title = ' Tiêu đề bình luận ';

Nhưng nếu bạn gọi từ 1 hàm trong đối tượng thì bạn có thể sử dụng từ khóa this

Ví dụ:

var Comment = {

title : '';

addComment = function(){

this.title = 'Tiêu đề bình luận';

}

}

b/ Lấy giá trị của thuộc tính

Để lấy giá trị của thuộc tính thì ta làm tương tự như cách thao tác với biến

var title = Comment.title;

Vẫn sử dụng this nếu gọi từ 1 hàm trong đối tượng

Ví dụ:

var Comment = {

title : '';

addComment = function(){

var title = this.title;

}

}

c/ Gọi phương thức

Tương tự thuộc tính, triển bình thường

comment.addComment();

Gọi trong hàm của đối tượng

var Comment = {

title = '';

addComment : function(){

this.validateComment();

},

validateComment : function(){

//Do Something

}

}

3/ Mảng đối tượng - Đối tượng chứa đối tượng

Mỗi đối tượng (object) trong Javascript có thể chứa các đối tượng khác hoặc một mảng có thể chứa các đối tượng.

a/ Đối tượng chứa đối tượng

Ví dụ 1 : gom các thuộc tính của Comment vào đối tượng info

var Comment = {

info : {

title : "",

content : "",

email : "",

fullname : ""

}

}

Ví dụ 2 : gom các phương thức hàm của Comment vào 1 đối tượng func

var Comment = {

func : {

addComment : function(){

//Do Something

},

validateComment : function(){

//Do Something

}

}

}

Lúc này để truy xuất tới các thuộc tính và phương thức này ta chỉ việc sử dụng dấu chấm và bổ sung thêm một cấp nữa.

Comment.info.title="Đây là 1 comment";

Comment.func.addComment();

4/ Mảng chứa đối tượng

Để gán giá trị là 1 đối tượng vào 1 mảng cũng tương tự như cách gán các giá trị bình thường khác.

content = '',
