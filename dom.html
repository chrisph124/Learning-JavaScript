DOM là viết tắt của chữ Document Object Model(Mô hình các đối tượng trong tài liệu HTML)

Trong mỗi thẻ(tag) của HTML sẽ có những thuộc tính(properties) và sự phân cấp cha-con với các thẻ khác. Sự phân cấp này đc gọi là các Selector và trong DOM sẽ có nhiệm vụ xử lý các vấn đề như thay đổi thuộc tính của thẻ, cấu trúc HTML của thẻ

Các thể loại của DOM trong HTML

Việc xử lý DOM trong HTML rất phức tạp nhưng cũng rất đa dạng vì javascript cung cấp rất nhiều phương thức, đối tượng và mỗi thành phần đều đảm nhận 1 nhiệm vụ riêng biệt để tiện sử dụng

Danh sách chia nhóm DOM :

  1. DOM document : lưu trữ toàn bộ các thành phần trong tài liệu website
  2. DOM element : truy xuất tới thẻ trong HTML thông qua các thuộc tính như tên của id, class
  3. DOM HTML : thay đổi giá trị nội dung, thuộc tính của các thẻ HTML
  4. DOM CSS : thay đổi các định dạng trong CSS của thẻ HTML
  5. DOM Event : gán các sự kiện vd như onClick(), onLoad() vào các thẻ HTML
  6. DOM Listener : lắng nghe các sự kiện tác động lên thẻ HTMl nào đó
  7. DOM Navigation : dùng để quản lý, thao tác với các thẻ HTML, thể hiện mối quan hệ cha-con của các thẻ HTMl
  8. DOM Node,Nodelist : thao tác với HTML thông qua đối tượng(object)
