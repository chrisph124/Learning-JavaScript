CSS bản chất cũng là 1 thuộc tính thẻ của HTML nhưng không thể sử dụng DOM HTML để thay đổi được mà phải thông qua 1 đối tượng đặc biệt khác là Style

Style chứa tất cả các thuộc tính của CSS và rất dễ thao tác với chúng bằng cú pháp riêng, có 2 thao tác với Style là thiết lập CSS và lấy giá trị CSS.

Cú pháp ví dụ :

document.getElementById('IDname').style.cssName = "cssAttribute"
